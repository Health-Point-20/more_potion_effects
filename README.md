## 📖 模组简介

More Potion Effects 是一个为 Minecraft 添加了丰富新状态效果和相关 enchantments 的模组。它扩展了药水酿造机制，并支持完整的自定义配置。


这个模组添加了超过 **80 种新状态效果**和 **11 种新 enchantments**，以及**自定义酿造系统**。
-注：这仅是此模组1.21.1版本的内容，当前尚不完善。

## ✨ 主要特性

### 🎯 状态效果

#### 正面效果（部分）
- **生命静止** - 所有伤害和治疗效果延迟到效果结束
- **不死** - 受到致命伤害时复活并恢复生命值
- **免疫** - 每级提供负面效果抗性，V 级免疫所有负面效果
- **真伤** - 将部分伤害转换为无视护甲的真实伤害
- **强心** - 放大治疗效果百分比

#### 负面效果（部分）
- 略

#### 中性效果（部分）
- **锁定** - 防止效果被移除
- **随机效果** - 随机获得正面/负面/中性效果


### 🧪 自定义酿造系统

- 支持通过 JSON 配置添加自定义酿造配方
- 药水可以同时拥有多个状态效果
- 可自定义药水名称和颜色
- 首次启动时自动生成配置文件夹和示例配方
- 合成的药水自动添加到创造模式物品栏
- 兼容 JEI 查看自定义配方

### 🔧 命令

```
/force_effect 或 /fe
```
强制效果管理命令，支持强制添加/移除效果（无视免疫）

### ⚙️ 配置选项

提供广泛的配置选项（`more_potion_effects.toml`）：
- 配置随机效果排除列表
- 调整特定效果参数
- 设置特定效果的视觉行为
